# English Words Quiz for 1st Grade

An interactive web app that helps first-grade students learn basic English vocabulary in a fun and engaging way.

Each card displays an image and a Ukrainian word. On click, the card flips to show the English translation along with its phonetic transcription. Users can also listen to correct pronunciation using speech synthesis.

## Features

- Interactive flip cards
- English pronunciation using SpeechSynthesis API
- Phonetic transcription loaded dynamically from dictionary API
- Category-based navigation (Animals, Food, etc.)
- Responsive and kid-friendly UI

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, animations)
- JavaScript (ES6+, async/await, modules)
- Axios (for API requests)
- SpeechSynthesis Web API

## Architecture

- Modular file structure
- Separation of logic:
  - data layer
  - UI rendering
  - API interaction
- DOM manipulation without innerHTML for better performance and security
