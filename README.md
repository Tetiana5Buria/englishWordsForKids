# English Words Quiz for 1st Grade

A simple interactive quiz to help first-grade students learn English words.
Each card shows a picture and the word in Ukrainian. Click the card to flip it and see the English translation with its transcription.
You can also listen to the pronunciation using the built-in speech feature.

## Important Note

This project is **non-commercial** and was developed for personal and educational use only.

## Features

- Flip cards to reveal the English translation and transcription
- SpeechSynthesis for correct pronunciation
- Kid-friendly, colorful, and responsive design
- Easy to extend with more words and images

## Tech Stack

- HTML5 & CSS3
- JavaScript (ES6+)
- SpeechSynthesis API for audio
- Free images from [Icons8](https://icons8.com)
